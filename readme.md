# Wordle

Scraped Wayback Machine Wordle files from before the 12th Feb 2021 (before New York Times changed the domain to redirect and added a tonne of popups)

### Will this last?

This code is scraped meaning it still has the ability to work exactly the same as before. There's still new words each day. This is just an archive.

### Can I play?

You can! You can even play offline.

### Online

- [Visit here](https://fluteds.github.io/wordle)

### Offline

#### Method 1

- Download the repo as a zip folder
- Unzip and run index.html however you choose

#### Method 2 (Out of the Box)

- Download wordle.html
- Open wordle.html
